## Repository Migration

This repository has been renamed from `nzxt-web-integration-amc` to `nzxt-esc`.

## What Changed

- **Old Repository:** `nzxt-web-integration-amc`
- **New Repository:** `nzxt-esc`
- **Old URL:** `https://mrgogo7.github.io/nzxt-web-integration-amc/`
- **New URL:** `https://mrgogo7.github.io/nzxt-esc/`

## For Users

### If you have the old URL bookmarked:

The old URL will automatically redirect to the new location. No action needed.

### If you're using NZXT CAM:

Update your Web Integration URL to:
```
https://mrgogo7.github.io/nzxt-esc/
```

Or use the NZXT CAM protocol:
```
nzxt-cam://action/load-web-integration?url=https://mrgogo7.github.io/nzxt-esc/
```

## For Developers

### Cloning the Repository

**Old:**
```bash
git clone https://github.com/mrgogo7/nzxt-web-integration-amc.git
```

**New:**
```bash
git clone https://github.com/mrgogo7/nzxt-esc.git
```

### Updating Remote URL

If you already have the repository cloned:

```bash
git remote set-url origin https://github.com/mrgogo7/nzxt-esc.git
```

## Redirect Setup

The old repository (`nzxt-web-integration-amc`) will remain available with redirect pages to ensure backward compatibility. All old URLs will automatically redirect to the new location.
